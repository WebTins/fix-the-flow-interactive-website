# Buurtcampuskrant

Tijdens sprint 5 ben ik vooral bezig geweest met het uitbreiden van de website. Ik heb me ook gericht om een interactie toe te voegen voor een filter omdat het ook de bedoeling is dat de gebruikers/doelgroepen naar hun eigen doelgroep kunnen filteren. Verder heb ik ook de focus gelegd op de toegankelijkheid van de website omdat er veel kleuren worden gebruikt die overeen moeten komen met de kleurcontrast. Hierbij heb ik ervoor gezorgd dat alle Lees meer knoppen dezelfde stijling krijgen met dezelfde kleur. Hierdoor wordt het meer leesbaar voor de gebruiker en geeft het ook een mooie uiterlijk aan de website.

### Wat is de Buurtcampuskrant?
De Buurtcampuskrant is een digitale krant die iedereen kan lezen maar voornamelijk voor mensen die in Amsterdam wonen. Het doel is om iedereen op de hoogte te brengen wat er gebeurd in de verschillende buurten. De doelgroepen van de Buurtcampus zijn: Studenten, Bewoners, Wijkpartners en HvA medewerkers. De Buurtcampuskrant brengt dus mensen samen die ook meer te komen weten over andere buurten als zij er ge√Ønteresseerd in zijn.

### Vraag van de opdrachtgever
Maak een website die gebruikers kunnen gebruiken als een digitale krant waar ze nieuws kunnen lezen en kunnen filteren op doelgroep. De digitale krant moet dezelfde uiterlijk en gevoel geven zoals de fysieke krant die elke 1,5 jaar wordt uitgebracht.

Link naar de website: https://webtins.github.io/fix-the-flow-interactive-website/
